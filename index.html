📌 Overview

ReconX is your Swiss Army Knife for network reconnaissance. Designed for ethical hackers, red teamers, and security researchers, it unifies multiple OSINT and scanning utilities into a clean, fast, and interactive Streamlit interface.

"Silent, swift, and powerful — ReconX simplifies deep network analysis without making noise."

🚀 Features

🔍 Port Scanner

Fast multithreaded port scan with service detection and vulnerability hints.

🧠 OS Fingerprinting

Estimate the target system's OS based on TCP/IP response signatures.

🌐 Whois Lookup

Get detailed ownership data of domains/IPs using public WHOIS services.

🔄 Reverse DNS

Reveal domains associated with given IPs for better context and analysis.

📡 DNS Enumeration

Uncover DNS records including A, MX, NS, CNAME, TXT to map domain structure.

📖 Usage Guide

  • ✅ Enter target IP or domain for each module
  • ⚙️ Click the respective action button (Scan, Lookup, etc.)
  • 📊 View rich results directly in your browser

🧬 How It Works

Each module uses Python-native or public libraries/APIs to perform passive or active recon. OS fingerprinting works via TCP handshake responses, WHOIS through RDAP, and DNS queries use `dnspython`.

⚙️ Installation

git clone https://github.com/atharvbyadav/ReconX.git
cd ReconX
pip install -r requirements.txt
streamlit run ReconX.py

🔓 ReconX will open in your browser automatically.
