🔍 Overview

ReconX is a modern, intuitive, and lightweight recon toolkit designed for ethical hackers, penetration testers, and cybersecurity enthusiasts. Powered by Python and Streamlit, it enables deep scanning, fingerprinting, and DNS intelligence with a simple UI.

🚀 Features

🔹 Port Scanning

Fast multi-threaded scanner for detecting open ports, service banners, and vulnerabilities.

🔹 OS Fingerprinting

Infers whether the target system is Linux or Windows using passive techniques.

🔹 WHOIS Lookup

Retrieve domain/IP ownership data from public registries.

🔹 Reverse DNS Lookup

Maps an IP address to a possible domain name using DNS records.

🔹 DNS Enumeration

Extracts DNS records like A, MX, NS, CNAME, TXT for infrastructure analysis.
