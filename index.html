Live App GitHub Repo

Overview

ReconX is designed to be both powerful and simple. With one click, perform port scans, OS detection, WHOIS lookups, reverse DNS and DNS enumeration — all within a clean, interactive Streamlit UI.

Whether you're a red-teamer mapping infrastructure or a security researcher gathering intelligence, ReconX helps you get actionable data fast and responsibly.

Features

Port Scanning

Multi-threaded scanning (50 threads) to detect open ports, banners, and outdated services quickly.

OS Fingerprinting

Infers if the target is Linux/Unix or Windows using TCP handshake analysis.

WHOIS Lookup

Pull registrar, owner, and domain date info directly from public WHOIS servers.

Reverse DNS

Translate IP to domain names to uncover hidden infrastructure associations.

DNS Enumeration

Fetch A, AAAA, MX, NS, TXT, CNAME records to build a robust domain profile.
